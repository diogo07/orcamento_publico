Introdução

Para realizar requisições na parte de receitas é necessário a passagem de alguns parâmetros, dentre eles está o código de cada municipio, que é exemplificado na seção municipios. Essa API fornece três tipos de consultas para a parte de receitas orçamentárias, que são: receitas anuais por município; receitas anuais por classificação e município e; receitas anuais por função e município. Abaixo serão demonstrados alguns exemplos destas consultas.

Receitas Anuais por Município

Para esse tipo de consulta necessita-se do código do município, no exemplo abaixo, foi utilizado o código 2512309, do município de Princesa Isabel - PB.

Requisição: http://127.0.0.1:8000/api/receita/municipio/2512309

Receitas por Município, Classificação e Ano

Esse tipo de consulta requer a passagem de dois parâmetros, o primeiro é o código do município, que é exemplificado na seção municípios. Como exemplo, será usado o código da cidade de Princesa Isabel - PB, que corresponde a 2512309, o segundo parâmetro é o ano da receita, para este exemplo será utilizado o ano de 2017. Veja abaixo como ficou a consulta:

Requisição: http://127.0.0.1:8000/api/receita/classificacao/municipio/2512309/ano/2017

Resposta:

[

{

"municipio": "Princesa Isabel",

"uf": "PB",

"regiao": "Nordeste",

"ano": 2017

"receitas": [

{

"classificacao": "Receitas Brutas Realizadas"

"valor": 44942716.42

{

"classificacao": "Deduções - FUNDEB"

"valor": 3200825.38

},

]

}

]
