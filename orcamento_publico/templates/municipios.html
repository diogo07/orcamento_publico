Introdução

As consultas por municípios podem ser realizadas de duas formas:

  • Por nome: nessa opção, pode-se passar o nome completo do município, ou parte dele. É importante destacar que a requisição não aceita espaços, dessa forma, caso o nome do município contenha mais de uma palavra, deve-se inserir um underline entre as palavras, para a API reconhecer o espaço existente. Por exemplo, para buscar a cidade de Princesa Isabel, deverá ser escrito Princesa_Isabel.
  • Por código: a busca por código requer o conhecimento prévio do código do município, dessa forma, para facilitar o acesso foi padronizado o código do município com o código definido pelo IBGE.

Caracteres Especiais

As requisições na API não suportam a passagem de caracteres especiais como acentos e o cedilha, por esse motivo, tornou-se necessário a padronização de identificadores para que a API reconheça esse tipo de caractere. Na tabela abaixo, estão os simbolos que devem ser substituidos pelos caracteres especiais, bem como um exemplo de como funciona esse processo.

CARACTERE SIMBOLO
ã __tila__
õ __tilo__
â __circunflexoa__
ê __circunflexoe__
ô __circunflexoo__
á __agudoa__
é __agudoe__
í __agudoi__
ó __agudoo__
ú __agudou__
ç __cedilha__

EXEMPLO: Para buscar o município de São Paulo, deve-se enviar a palavra S__tila__o_Paulo.
